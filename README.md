# voidkde
post install kde cuztom script


# Features
- Add NonFree & Multilib Repositories
- Remove Blotware
- Add Laptop Utilities
- Up IMportant Services And Demons

# Requirements
- VOID OS
- Terminal With Bash 

# Installation
```
- sudo su
- git clone http://github.com/callmezatiel/voidkde
- cd voidkde
- chmod +x voidkde.sh
- sudo ./voidkde.sh

```
